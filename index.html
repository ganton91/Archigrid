Instructions
What is ArchiGrid

ArchiGrid is a grid-based architectural drawing tool. Every cell is 5×5 cm. You paint cells with colors to represent spaces, walls, openings, or any programmatic element you define.

The canvas is infinite — pan freely and zoom in or out to work at any scale.

Layers

Organize your drawing into layers — one per floor, program type, or phase. Each layer has its own color cells, opacity, and visibility. The active layer always renders on top.

Drag the handle (⠿) to reorder. Click the dot (●) to toggle visibility. Click (◉) to solo a layer and hide all others temporarily.

Brush

Set brush size in X and Y independently (minimum 5cm = 1 cell). Paint with left click, erase with right click. The brush preview follows your cursor on the canvas and rulers.

Choose from 14 palette swatches or fine-tune with the HSB sliders. Use the eyedropper to pick a color directly from the canvas.

Scenes

Import reference images (floor plan scans, site maps, photographs) as Scenes. Each scene can be moved, rotated, and scaled freely on the canvas.

Use Set Scale to calibrate a scene to real-world dimensions: click two points on the image, then enter the real distance in meters. All subsequent measurements will be accurate.

Calculations

Build formulas using layer properties as building blocks. Insert badges for Area, Cell Count, or per-color variants. Combine them with arithmetic operators (+, −, ×, ÷) typed freely between badges.

Reference other Calculations inside a formula using Insert Calc. Results update live as you draw. Circular references are detected automatically and shown as ERR.

Export / Import

Export saves the entire project as a .json file — all layers, cells, scenes (including images as base64), calculations, palette, brush settings, and viewport position.

Import restores a previously exported project completely. The filename is remembered and shown in the header for reference.
